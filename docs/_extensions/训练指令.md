当前仓库训练指令
可用任务
任务 ID	说明
Isaac-Velocity-Flat-EVA02-v0	EVA02 平坦地形速度跟踪
Isaac-Velocity-Rough-EVA02-v0	EVA02 崎岖地形速度跟踪
Isaac-Velocity-Flat-EVA02-CTS-v0	EVA02 CTS（师生协同）平坦地形
Isaac-Velocity-Rough-EVA02-CTS-v0	EVA02 CTS 崎岖地形
1. 标准 RSL-RL 训练
    平坦地形训练（有界面）
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02.py
    无头模式（更快）
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02.py --headless
    崎岖地形训练
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02.py \
        --task Isaac-Velocity-Rough-EVA02-v0

    指定环境数量 + 录制视频
./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02.py \
    --num_envs 4096 --video
2. CTS 训练（师生协同）
    CTS 平坦地形
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02_cts.py

    CTS 崎岖地形
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02_cts.py \
    --task Isaac-Velocity-Rough-EVA02-CTS-v0

    无头模式
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train_eva02_cts.py --headless
3. 推理/回放
    标准回放
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/play.py \
        --task Isaac-Velocity-Flat-EVA02-v0 \
        --checkpoint logs/rsl_rl/<实验名>/model_<步数>.pt
    CTS 回放
    ./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/play_cts.py \
        --task Isaac-Velocity-Flat-EVA02-CTS-v0 \
        --checkpoint logs/cts/<实验名>/model_<步数>.pt
4. 查看机器人模型
    ./isaaclab.sh -p scripts/view_eva02.py

常用参数
参数	说明	默认值
--task	任务 ID	Isaac-Velocity-Flat-EVA02-v0
--num_envs	并行环境数	配置文件中的值 (4096)
--headless	无头模式（无渲染）	False
--video	录制训练视频	False
--video_length	视频长度（步）	200
--video_interval	录制间隔（步）	2000
--seed	随机种子	None（随机）
--checkpoint	模型检查点路径（play）	必需
